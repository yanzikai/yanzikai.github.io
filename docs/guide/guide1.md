## 精度问题 
后端返回超出长度的数值，转化之后出现精度问题
``` js
> var res = '{"num":123456789123456789}'
> JSON.parse(res)
< {num: 123456789123456780}
```
使用正则在JSON转化之前把数值转换成字符串
``` js
> var res = '{"num":123456789123456789}'
> var res1 = res.replace(/(\d+)/g, '"$&"')
> res = JSON.parse(res1)
< {num: '123456789123456789'}
```
